Useful Commands

bundle exec jekyll serve --livereload --incremental

Builds your site and makes it available on localhost:4000, with the site being incrementally updated on saved changes.

Adding a Topic

  1. In config.yml, add it to the collections hash and set output: true
  2. Create a _topic_name folder in root, this will store the pages for sub-topics, then create any sub-topics you know you'll need
  3. Manually add the layout to the frontmatter for that page
  4. Create the layout, can copy everything except the name of the nav from any other layout
  5. Create the nav, can copy and just ctrl+f the old topic. Then add it to the layout.

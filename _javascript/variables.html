Variable Declarations

const
Declares an immutable variable which will raise an error if modified. Is block scoped to the block it was declared in, and is not hoisted. Declare memory hogging objects with this inside a block to garbage collect the object when the block is done executing.
let
Declares a mutable variable which is block scoped and will not be hoisted. Declare memory hogging objects with this inside a block to garbage collect the object when the block is done executing.
var
Declares a mutable variable which is function scoped and will be hoisted for access if declared after it's used. As it's function scoped, using var in a for loop or conditional actually declares that variable in the global scope, or the function scope if its within one.
